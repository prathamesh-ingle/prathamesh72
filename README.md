## ✨ Key Features

This portfolio isn't just a static website; it's an interactive experience built with modern web technologies:

* **Cinematic GSAP Animations:** Features complex `ScrollTrigger` animations, including parallax scrolling, squeezing hero sections, and staggered 3D reveals.
* **Integrated 3D Graphics:** Utilizes `React Three Fiber` and `Three.js` to render a fully interactive, hovering 3D jet that physical "dives" and pins to the screen as you scroll.
* **Glassmorphic UI Design:** A sleek, dark/light theme hybrid featuring frosted glass cards, ambient glowing orbs, and dynamic CSS gradients.
* **Liquid Scrolling:** Buttery smooth scrub animations that tie physical element movement directly to the user's scroll wheel.
* **Fully Responsive:** Carefully crafted `matchMedia` queries ensure the heavy animations run flawlessly on desktop while remaining smooth and optimized on mobile devices.

## 🛠️ Tech Stack Built With

* **Frontend Framework:** React.js (Vite/CRA)
* **Styling:** Tailwind CSS
* **Animations:** GSAP (GreenSock) & Framer Motion
* **3D Rendering:** React Three Fiber, Three.js
* **Icons:** React Icons (FontAwesome, SimpleIcons, Lucide)
* **Scroll Effects:** AOS (Animate On Scroll) & GSAP ScrollTrigger
